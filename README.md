# Phishing Awareness Simulation

## Introduction
Phishing is a social engineering technique used to trick users into revealing sensitive information through fake emails or messages.

## Objective
To study common phishing techniques and understand how users can identify and avoid them.

## Social Engineering Techniques Used
- Urgency
- Fake authority
- Spoofed email addresses
- Malicious links

## Scenario Description
This exercise uses a simulated phishing email claiming to be from an IT support team requesting password verification.

## Impact
Successful phishing attacks can lead to data breaches and account compromise.

## Prevention Methods
- Verify sender email addresses
- Avoid clicking suspicious links
- User awareness training
- Email filtering systems

## Conclusion
Understanding phishing techniques helps users reduce the risk of falling victim to social engineering attacks.

## Ethical Disclaimer
This is a simulated exercise for educational purposes only.
